# OpenGL 4.6 Type-Safe Wrapper

Это полнофункциональная типобезопасная обертка над OpenGL 4.6 на C++23, реализующая современный безопасный фреймворк для работы с графическим API с полной ориентацией на Direct State Access (DSA) и Approaching Zero Driver Overhead (AZDO) подходы.

## Особенности

- Полная типобезопасность через enum class
- Хендлы ресурсов без внутренней логики
- Глобальные функции для управления
- Современный безопасный API
- Поддержка всех функций OpenGL 4.6
- Поддержка расширений
- Ориентация на DSA и AZDO подходы
